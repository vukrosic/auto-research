| # | Name | Change | Size Estimate | Rationale |
|---|------|--------|---------------|-----------|

### Wave 4 — Day 5–7 (Mar 29–31)
**Type**: Full + Combination Explores
**Experiments**: 0-1 full run + combination explores
**GPU time**: ~13-20 hours

**If we have a validated winner** (>0.01 BPB at 4000 steps):
- Run 1 full (13780 steps, ~12.7 hr)
- Fill remaining time with combination explores

**If validates are marginal** (0.001-0.01 BPB):
- Skip full run
- Run combination experiments: stack 2+ independent winners
- Use remaining budget for more explores on new variants

**If nothing validated**:
- Pivot: close axes 1-8, open combination/hybrid approaches
- Run ~15 more explores on novel combinations
- Write knowledge on what doesn't work

- Final adjudication of all results
- Update knowledge files
- Write campaign progress report
